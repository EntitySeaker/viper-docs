### Purpose
Executes a program.

```bash
exec [PATH]
```

### Arguments
- PATH -- specify the path to the program you want to execute

### Examples
```bash
exec /usr/bin/Terminal.exe
```
Result:  
*Executes program `/usr/bin/Terminal.exe` which opens a terminal*
