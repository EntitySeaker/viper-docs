### Purpose
Shows all running processes on a computer.

```bash
ps
```

### Examples
```bash
ps
```
Result:
```
USER  PID   CPU     MEM    COMMAND      
----  ---   ---     ---    -------      
root  8455  0.0%    13.7%  kernel_task  
root  9723  0.0%    11.2%  Xorg         
volk  4335  0.0%    0.0%   Chat         
volk  2050  0.0%    8.6%   Terminal     
volk  1577  0.0%    0.0%   viper        
volk  1145  0.0%    8.6%   Terminal     
volk  4100  100.0%  0.0%   viper
```
