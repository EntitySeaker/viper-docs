<Aside type="caution">For security reasons, in order to save your settings you must create a file called `Viper.conf` in the user's home folder.</Aside>

### Purpose
Saves settings.

```bash
save-settings
```

### Examples
```bash
save-settings
```
Results:  
```
Settings saved!
```
