<Aside type="note">msfconsole is always started from the selected metaxploit.so library, which can be selected using the uselib command.</Aside>

### Purpose
Starts a listiner for incomming rshells.

```bash
msfconsole
```

### Examples
```bash
msfconsole
```
Result:  
*Started a reverse shell listiner*
