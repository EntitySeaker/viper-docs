### Purpose
Returns the version of a library.

```bash
scanlib [PATH]
```

### Arguments
- PATH -- specify the path to the library you want to know the version of

### Examples
```bash
scanlib /lib/init.so
```
Result:
```
Name: init.so
Version: 1.0.2
```
