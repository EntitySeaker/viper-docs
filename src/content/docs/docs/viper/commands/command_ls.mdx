### Purpose
Lists files inside a directory.

```bash
ls [PATH]
```

### Arguments
- PATH -- the path to a folder to list

### Examples
```bash
ls /home
```
Result:
```
permissions  user   group  name     
-----------  ----   -----  ----     
drwxrwx---   volk   volk   volk  
drwxrwxrwx   guest  guest  guest
```
