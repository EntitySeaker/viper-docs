### Purpose
Cracks an MD5 hash.

```bash
crack [HASH]
```

### Arguments
- HASH -- specify the hash to crack

### Examples
```bash
crack 63a9f0ea7bb98050796b649e85481845
```
Result:  
*Returns the password of the hash if found.*
