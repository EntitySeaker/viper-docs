### Purpose
Adds a variable to the list of variables.

```bash
addvar [KEY] [VAL]
```

### Arguments
- KEY -- choose a name for your variable
- VAL -- choose the value of the variable

### Examples
```bash
addvar ip 177.58.127.159
```
Results:  
*Added a new variable of "ip" with "177.58.127.159" as it's value.*  
*Using "ip" will now translate to "177.58.127.159"*
