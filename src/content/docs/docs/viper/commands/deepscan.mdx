### Purpose
Scans every IP behind a router.

```bash
deepscan [IP]
```

### Arguments
- IP -- the public IP or IP of the router you want to scan.

### Examples
```bash
deepscan 177.58.127.159
```
Results:  
```
192.168.0.1
192.168.0.2
192.168.0.3
10.0.19.1
172.16.20.1
```
