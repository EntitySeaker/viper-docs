### Purpose
Backs out to the previous session.

```bash
back
```

### Examples
```bash
back
```
Result:
```
———(root@viper)——[215.250.95.157]——[192.168.0.19]——[SHELL]
——$ back
Going back to the previous position.
———(volk@viper)——[215.250.95.157]——[192.168.0.19]——[SHELL]
```
