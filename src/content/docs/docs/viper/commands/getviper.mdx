### Purpose
Launches another instance of Viper, and transfers objects on exit.

```bash
getviper [PATH] (ARGS)
```

### Arguments
- PATH -- specify the path to the new viper instance
- (ARGS) -- specify any arguments you want to pass to the new viper isntance for example: `-a` for anonymous mode

### Examples
<Tabs>
  <TabItem label="Without args">
    ```bash
    getviper /home/guest/viper
    ```
    Result:  
    *Launched a new viper isntance that will carry over objects like libraries and sessions on exit.*
  </TabItem>
  <TabItem label="With args">
    ```bash
    getviper /home/guest/viper -a
    ```
    Result:  
    *Launched a new viper isntance in anonymous mode that will carry over objects like libraries and sessions on exit.*
  </TabItem>
</Tabs>
