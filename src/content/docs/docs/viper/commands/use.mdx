### Purpose
Uses a selected target.

```bash
use [INDEX]
```

### Arguments
- INDEX -- the index of the session found by using the [targets](/docs/viper/commands/targets/) command

### Examples
```bash
use 0
```
Result:
```
———(volk@viper)——[215.250.95.157]——[192.168.0.19]——[COMPUTER]
```
