### Purpose
Scan a host, or a randomly selected target to discover open ports and basic service information.

```bash
nmap [IP:RANDOM]
```

### Arguments
- IP -- single IPv4 address (example: 177.58.127.159)
- RANDOM -- a random IPv4 address (example: random)
