### Purpose
Returns domain information from an IP.

```bash
whois [IP]
```

### Arguments
- IP -- specify the IP you want to know the domain information of

### Examples
```bash
whois 5.155.150.245
```
Result:
```
Domain name: www.efacujeh.info
Administrative contact: Joya Kohl
Email address: Kohl@efacujeh.info
Phone: 260532383
```
