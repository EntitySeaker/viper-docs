### Purpose
Returns the IP of a domain.

```bash
nslookup [DOMAIN]
```

### Arguments
- DOMAIN -- specify the domain you want to know the IP of

### Examples
```bash
nslookup www.efacujeh.info
```
Result:
```
IP: 5.155.150.245
```
