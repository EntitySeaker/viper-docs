<Aside type="note">Only the root user can remove a user from a group.</Aside>

### Purpose
Removes a user from a group.

```bash
delgroup [USER] [GROUP]
```

### Arguments
- USER -- specify the user you want to remove from the group
- GROUP -- specify what group you want to remove from the user

### Examples
```bash
delgroup volk root
```
Result:  
*Removed volk from the root group.*
