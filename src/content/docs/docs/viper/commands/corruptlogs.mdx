<Aside type="note">Write permissions to the `/var` directory are required, so root permissions is likely needed.</Aside>
<Aside type="tip">It's best practice to run this command every now and then on a target incase a new logfile got created.</Aside>

### Purpose
Corrupts the logfile stored at `/var/system.log`.

```bash
corruptlogs
```

### Examples
```bash
corruptlogs
```
Result:  
*The logfile is now corrupted*
