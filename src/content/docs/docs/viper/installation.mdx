### Steps
<Steps>

1. Open your browser and go to: https://git.viper-guild.com/Viper/viper

2. On the page, locate and copy the full contents of the file named **viper.src**.

3. Paste the copied source into your Grey Hack code editor.

4. Click the **Compile** button in the editor.

5. Once compilation finishes, load or run Viper according to your in-game workflow.

</Steps>
